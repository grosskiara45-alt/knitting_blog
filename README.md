## Introduction
Overview:

A Code:You assignment in the Web Development pathway utilizing HTML, CSS, and more web development tools to create a small project of our choice.

Project Overview:
As someone who is currently learning both knitting and web development, I decided to create my own blog site that will include blog posts, an about me page, and a contact page. Through this project, I hope to not only track my knitting journey but also practice the skills that I am learning through Code:You.
